## 1. 食材图标风格定义

### 设计原则
- **风格**：Kawaii / Q版卡通风格 — 圆润线条、柔和阴影、微表情
- **一致性**：统一描边宽度（2px）、统一光源（左上 45°）
- **辨识度**：缩至 32px 仍可辨识
- **背景**：透明

### 技术规格

| 属性 | 规格 |
|------|------|
| 尺寸 | 128×128px 设计稿，使用时 64×64px |
| 格式 | SVG 优先，PNG@2x 备选 |
| 描边 | 2px，颜色 `#5D4037` |
| 阴影 | 底部 2px，`rgba(0,0,0,0.15)` |

## 2. 自定义食材图标 (v1.1 新增)

### 菜篮图标 (Custom Ingredient Basket)
- **形态**：一个编织竹篮/菜篮，略带卡通比例（圆胖）
- **特效**：篮子周围有微弱的金色光粒子环绕，暗示"什么都能放进来"
- **大小**：与普通食材图标一致 128×128px
- **放置位置**：右栏食材区底部，独立一行居中

### 自定义食材实例图标 (Light Orb)
- **形态**：半透明发光椭圆球体，内部有食材名称文字
- **颜色**：金色/琥珀色光晕（`#F0D58C` → `#C8956C` 渐变）
- **用途**：当用户输入自定义食材后，该食材在食材区中以此图标形态展示

## 3. 核心场景素材 (v1.1 修订)

### 3.1 烹饪锅 (The Cooking Pot)

**v1.1 重点修订**：从「魔法坩埚」改为「现代厨房铸铁锅 + 微魔法点缀」

**视觉描述**：
- **形状**：圆底铸铁汤锅（类 Le Creuset 珐琅锅造型），双耳把手，放在燃气灶台上
- **颜色**：深灰铸铁色（`#3D3D3D`），锅沿有一圈淡金色光晕描边（唯一的魔法元素）
- **灶台**：不锈钢/黑色玻璃灶面，干净整洁
- **状态**：
  - **待机**：锅沿微弱金色光晕缓慢呼吸
  - **烹饪中**：锅内蒸汽上升，食材翻滚可见，锅沿光晕加强
  - **成功**：金色星光粒子从锅口升起 → 淡出
  - **失败**：黑烟 → 锅盖弹起 → 烟雾散去

### 3.2 厨房台面 / 出单台面 (The Kitchen Counter)

**视觉描述**：
- **材质**：浅色大理石/石英石台面（`#F5F0E8`），带有浅灰色纹理
- **环境**：台面后方是白色瓷砖墙面，上方有暖色吊灯
- **装饰**：台面角落有一小盆绿植、一个木质胡椒研磨器
- **风格**：现代厨房杂志封面感，整洁明亮

### 3.3 出单纸条 (Recipe Ticket)

**视觉描述**：
- 仿餐厅小票样式：窄长矩形，顶部和底部锯齿状撕裂边缘
- **底色**：微黄/米白（`#FFF8E7`）像真纸
- **文字**：深棕色（`#3D3024`）
- 菜名居中加粗大字
- 翻车率用厨师红大号字体（`#E85D4A`）
- 食材清单等宽字体对齐
- 底部虚线分割
- 微旋转（-3° ~ +3°）散落效果

### 3.4 页面背景

**v1.1 修订**：
- **主烹饪页**：深色干净背景（`#1C1C28`），不再有砖墙和大量厨具装饰。背景保持极简，让三栏内容成为视觉焦点
- **出单台面页**：浅色大理石台面背景，上方有模糊的白色瓷砖墙纹理

## 4. 动画概念描述

### 4.1 食材飞入（右栏 → 中间锅）
- 触发：用户在右栏点击食材
- 效果：图标副本沿抛物线从右往中飞向食材槽
- 时长：~500ms
- 音效建议："pop" + "whoosh"

### 4.2 烹饪动画（成功路径）
```
阶段1: 食材从槽中逐一跳入锅中 (各 200ms)
阶段2: 锅内蒸汽加速，食材翻滚（循环动画）
阶段3: 金色星光粒子从锅口升起 → 淡出 → 切换至台面页
```

### 4.3 烹饪动画（失败路径）
```
阶段1: 锅冒黑烟
阶段2: 锅盖弹起 + 黑色烟雾喷出
阶段3: 烟雾散去，恢复正常 → 弹出警告
```
- 时长：~1.5s

### 4.4 纸条飘落
- 1-3 张纸条从上方飘落，spring 物理，轻微摇摆
- 各纸条延迟 200ms 间隔
- 散落后带微旋转
