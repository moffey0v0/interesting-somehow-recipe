# UI/UX 设计规范 — Phase 1.1 (修订版)

> 项目：大概有趣菜谱 / Interesting Somehow Recipe  
> 版本：v1.1 | 2026-03-09（基于 p1 修订）

---

## 1. 设计理念 (v1.1 修订)

**核心关键词**：整洁 · 现代厨房 · 卡通点缀 · 幽默  
**视觉隐喻**：一间明亮干净的现代厨房，白色瓷砖台面、不锈钢厨具、原木隔板——但仔细看会发现一些有趣的魔法细节：按钮上刻着淡淡的符文、锅边偶尔闪过微光、食材飞入时有一丝星光粒子。

**🚫 避免**：过多的紫色/绿色魔药感、坩埚外观、阴暗炼金氛围。整体必须像「愿意在这里做饭吃东西」的厨房。

## 2. 色彩体系 (v1.1 修订)

### 主色板

| 用途 | 颜色名 | HEX | 说明 |
|------|--------|-----|------|
| 背景-深 | 厨房夜幕 | `#1C1C28` | 主背景，深色但干净 |
| 背景-中 | 石板灰 | `#2A2A3A` | 面板/卡片底色 |
| 背景-浅 | 大理石白 | `#F5F0E8` | 台面、小纸条底色 |
| 主色 | 暖木金 | `#C8956C` | 原木按钮、标题高亮 |
| 主色-深 | 橡木棕 | `#8B6B4A` | 边框、icon 描边 |
| 强调色-1 | 厨师红 | `#E85D4A` | CTA 按钮、翻车率指示 |
| 强调色-2 | 香草绿 | `#5CB85C` | 成功状态、安全标识 |
| 点缀色 | 符文金光 | `#F0D58C` | 发光特效、符文纹路（微量使用） |
| 创造模式 | 创意橙 | `#F39C12` | 创造模式新增食材高亮 |
| 文字-主 | 近白 | `#F0EDE6` | 正文（深色背景上） |
| 文字-次 | 暖灰 | `#A0A0A0` | 辅助说明 |
| 文字-深 | 深棕 | `#3D3024` | 正文（浅色背景上，如纸条） |

> **v1.1 变更**：去除了大面积紫色（魔法紫），改用暖棕/金色系为主；危险色改为更厨房感的「厨师红」；强调色改为「香草绿」；新增「大理石白」用于台面/纸条等浅色场景。

### 渐变

- **锅体光效**：`radial-gradient(circle, rgba(240,213,140,0.3) 0%, transparent 70%)`（微弱金色光晕）
- **背景**：`linear-gradient(180deg, #1C1C28 0%, #2A2A3A 100%)`

## 6. 组件交互逻辑 (v1.1 修订)

### 6.1 食材选择交互
1. 用户在**右栏**点击食材图标
2. 图标播放「弹跳」微动画（scale 1.0 → 1.2 → 1.0）
3. 食材图标副本以抛物线轨迹飞向**中间食材槽**
4. 食材槽中对应食材数量 +1，带 `pop` 入场动画
5. 再次点击 → 数量 +1
6. 点击食材槽中的食材 → 数量 -1（若为 0 移除）

### 6.2 自定义食材交互 (v1.1 新增)
1. 右栏底部有一个**发光菜篮图标**（✨🧺），带有微弱的金色脉冲光效
2. 点击后弹出输入框 Modal：
   - 输入框 placeholder："输入任何食材... 哪怕是月亮"
   - 输入完成后点击确认
3. 自定义食材以**发光光球图标**（✨🔮）显示在食材区中
4. 行为与普通食材一致：点击飞入锅中，可重复添加

### 6.3 烹饪触发交互
1. 锅处于「待机」状态：微弱金色光晕呼吸效果
2. 用户点击锅
3. **安全校验**：
   - ✅ 通过 → 烹饪动画（食材在锅里跳跃翻滚）
   - ❌ 不通过 → 锅冒黑烟 → 爆炸 → 弹出警告框
4. 烹饪动画作为 Loading（3-8s）
5. AI 返回后 → 过渡到出单台面页

### 6.4 结果展示交互
1. 画面过渡到「出单台面」（浅色大理石背景）
2. 1-3 张小纸条从上方「飘落」，带锯齿边缘，微旋转
3. 悬停查看概要（菜名、翻车率、食材摘要）
4. 点击纸条 → 弹出详细菜谱海报（居中弹窗）
5. 海报内提供「保存为图片」按钮

### 6.5 模式切换
- **生存 → 创造**：界面暖色调微增（橙色滤镜 +5%）
- **创造 → 生存**：反向
- 切换时长 300ms，ease-in-out

## 8. 动效规范

| 动效 | 时长 | 缓动 | 说明 |
|------|------|------|------|
| 食材弹跳 | 200ms | spring(300) | 点击反馈 |
| 食材飞行 | 500ms | ease-out | 右栏→中间锅 |
| 锅呼吸光效 | 2500ms | ease-in-out | 微弱金色光晕 |
| 烹饪翻滚 | 800ms/圈 | linear | 循环至 AI 响应 |
| 锅冒烟/爆炸 | 1000ms | ease-out | 一次性 |
| 纸条飘落 | 600ms | spring(damping: 15) | 入场 |
| 页面过渡 | 400ms | ease-in-out | 厨房→台面 |
| 菜篮发光 | 2000ms | ease-in-out | 循环金色脉冲 |
