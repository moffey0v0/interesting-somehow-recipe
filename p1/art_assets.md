## 1. 食材图标风格定义

### 设计原则
- **风格**：Kawaii / Q版卡通风格 — 圆润线条、柔和阴影、微表情（可爱的眼睛/脸颊红晕）
- **一致性**：所有图标统一描边宽度（2px）、统一光源方向（左上 45°）
- **辨识度**：食材的形状特征必须一眼可辨，即使缩小到 32px 仍可识别
- **背景**：透明背景

### 技术规格

| 属性 | 规格 |
|------|------|
| 尺寸 | 设计稿 128×128px，使用时缩放至 64×64px |
| 格式 | SVG（优先，矢量可缩放）、PNG@2x（备选） |
| 色域 | sRGB |
| 描边 | 2px，颜色 `#5D4037`（深棕色） |
| 阴影 | 底部 2px 投影，颜色 `rgba(0,0,0,0.15)` |

## 2. 核心场景素材

### 2.1 魔法锅 (The Magic Pot)

**视觉描述**：
- 形状：圆底大铁锅 + 三脚架底座，锅口冒出紫色/绿色魔法蒸汽
- 细节：锅身有炼金符文纹路，锅底有微弱火焰
- 状态变化：
  - **待机**：微弱发光，蒸汽慢速上升
  - **烹饪中**：强光脉冲，蒸汽加速，锅内可见食材翻滚
  - **成功**：金色星光爆发，锅口飘出美食香气（用星星粒子表示）
  - **失败**：黑烟滚滚 → 火焰喷发 → 💥爆炸 → 锅身变黑（然后恢复）

### 2.2 厨房台面 (The Kitchen Counter)

**视觉描述**：
- 木质纹理台面，温暖的橡木色调
- 台面上有几处装饰：小摆件、调料瓶、一根勺子
- 用于结果展示场景的背景

### 2.3 出单纸条 (Recipe Ticket)

**视觉描述**：
- 仿餐厅小票样式：窄长矩形，顶部和底部有锯齿状撕裂边缘
- 底色：微黄/米白色（像真实的热敏纸）
- 文字排版：
  - 菜名居中加粗大字
  - 翻车率用红色大号字体
  - 食材清单用等宽字体对齐
  - 底部有虚线分割
- 轻微旋转角度（每张纸条随机 -3° ~ +3°）模拟真实散落效果

### 2.4 页面背景

**视觉描述**：
- 整体是一个卡通厨房的远景
- 后方有一面砖墙，挂着几个锅具和各种厨房工具
- 上方有一盏温暖的吊灯发出暖黄光
- 氛围：整洁、温暖、带魔法气息（某些瓶罐会微微发光）

## 3. 动画概念描述

### 3.1 食材飞入动画

```
[食材图标] ── 点击 ──→ [弹跳缩放] ── 0.5s ──→ [抛物线飞行]
                         scale 1→1.2→1            ╭────╮
                                                    │     ╲
                                                    │      ╲
                                                  起点      落点(食材槽)
```

- 触发：用户点击食材图标
- 效果：图标副本沿抛物线飞向食材槽
- 时长：~500ms
- 音效建议：轻快的 "pop" + "whoosh"

### 3.2 烹饪动画（成功路径）

```
阶段1(准备): 锅开始抖动，发光增强
阶段2(投入): 食材从食材槽一个个跳入锅中 (每个200ms间隔)
阶段3(烹饪): 锅内旋涡效果，食材高速旋转 + 各种粒子特效
阶段4(完成): 金色光柱从锅内射出 → 淡出 → 场景切换至台面
```

- 总时长：3-8 秒（取决于 AI 响应速度）
- 阶段 3 为循环动画，可无限播放至 AI 返回结果

### 3.3 烹饪动画（失败路径）

```
阶段1: 锅开始抖动 (同成功路径)
阶段2: 锅冒黑烟，颜色变暗
阶段3: 火焰从锅口喷出 🔥
阶段4: 💥 爆炸！ — 全屏震动效果 + 碎片粒子
阶段5: 烟雾散去，锅变成焦黑色 (1秒后恢复正常)
```

- 触发条件：前端安全校验未通过
- 总时长：~2 秒
- 结束后弹出警告框说明不安全原因

### 3.4 纸条飘落动画

```
     ╔══════╗
     ║ 纸条1 ║  ← 从画面上方落入
     ╚══════╝
         ↓  (spring physics, 轻微左右摇摆)
    ╔══════╗
    ║ 纸条2 ║  ← 延迟200ms落入
    ╚══════╝
         ↓
   ╔══════╗
   ║ 纸条3 ║  ← 延迟400ms落入
   ╚══════╝
   最终散落在台面上，每张有轻微随机旋转角度
```
